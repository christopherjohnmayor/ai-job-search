Once you fork on GitHub, set YOUR fork as origin and re-add upstream:

    # Replace <your-github-handle> with your actual handle
    git remote set-url origin https://github.com/<your-github-handle>/ai-job-search.git
    git remote add upstream https://github.com/MadsLorentzen/ai-job-search.git
    git fetch --all
